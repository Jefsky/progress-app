项目进度管理系统 - 局域网部署

快速部署指南

按照以下步骤在局域网中部署项目进度管理系统：

打开命令提示符或 PowerShell
进入项目目录: cd d:\hello\progress-app
安装依赖: npm install
构建前端应用: npm run build:lan
启动 API 服务器: npm run server (新窗口)
启动前端服务器: npm run serve:static (新窗口)
注意: 确保您的防火墙允许 3000 和 8080 端口的访问，以便其他设备可以连接到您的服务器。

服务器地址

部署完成后，您可以通过以下地址访问应用：

本地访问

前端应用: http://localhost:8080

API 服务器: http://localhost:3000

局域网访问

前端应用: http://[您的IP地址]:8080

API 服务器: http://[您的IP地址]:3000
